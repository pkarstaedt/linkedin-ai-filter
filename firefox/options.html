This JSON will be sent as the request body for each post. The placeholder %post_test% will be replaced with the JSON-encoded text of the LinkedIn post. Your local LLM endpoint should respond with JSON in the form: { "linkedin_post_ai_score": VALUE }, where VALUE is an integer from 0 to 100.
Posts with AI score greater than or equal to this value will be hidden. Lower values hide more posts; higher values hide fewer.
